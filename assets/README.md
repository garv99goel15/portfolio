# Assets

Drop your résumé PDF here named exactly:

```
Garv_Goel_Resume.pdf
```

The **Download CV** buttons in the hero and the menu link to
`./assets/Garv_Goel_Resume.pdf`. Until the file exists, clicking shows a
friendly toast instead of a 404.

You can also add images here (e.g. an `og-image.png` for social sharing,
or project thumbnails) and reference them from `index.html`.
